Smart Calculator Help : Calculate integral and plot complicated chart

Theoretically, Smart Calculator is able to support any level integral calculation by reversely calling built-in integrate function. However, the internal integration tool only supports triple integration to avoid complexity. Using this tool, user needs to input the expression to be integrated, e.g. 2 + 3* x. Then user inputs the integrating variable name, in the example it is x. And then user types variable changing range, for example, from 3 + i to 100 -9i. The number of integration steps determines accuracy of the calculation. Lager number of steps leads to more accurate result however longer calculating time. Generally 100 integration steps have been enough. If the number of steps is zero or empty, Smart Calculator employs adaptive Gauss-Kronrod method to calculate integral which is more accurate for high-frequency functions. But it may be slow in some extreme cases.

Smart Calculator is also able to calculate indefinite integral. Note that not all the expressions can be indefinitely integrated. Moreover, some expressions may be too complicated to solve. In this case, Smart Calculator will throw an exception telling user that it cannot get result.

Smart Calculator has an internal tool to plot 2D chart. This tool supports at most eight 2D curves. User is able to add new curve by clicking "Add Curve" button or delete an existing curve by clicking "Delete Curve" button. For each curve, both X and Y values are a function of variable t whose value range is set by user. For example, t is from -50 to 50 and the length of step is 0.1, expression of X is 3 + t**2 and expression of Y is cos(t - 9). Please note that this plotting tool actually calls a built-in function named plot_2d_curves to draw charts. If user wants calculator to detect singular point(s) (i.e. x or y is infinite), step field should be left as blank so that auto step is choosen. After a chart is plotted, user is able to freely scale and shift the chart.

Smart Calculator has an internal tool to plot polar chart. This tool supports at most eight polar curves. User is able to add new curve by clicking "Add Curve" button or delete an existing curve by clicking "Delete Curve" button. For each curve, both r and θ values are a function of variable t whose value range is set by user. For example, t is from 0 to PI*2 and the length of step is PI/8, expression of r is cos(t) and expression of θ is t. Please note that this plotting tool actually calls a built-in function named plot_polar_curves to draw charts. If user wants calculator to detect singular point(s) (i.e. r or θ is infinite), step field should be left as blank so that auto step is choosen. After a chart is plotted, user is able to freely scale and shift the chart.

Smart Calculator has an internal tool to plot 3D chart. This tool supports at most four 3D surface curves. User is able to add new curve by clicking "Add Curve" button or delete an existing curve by clicking "Delete Curve" button. For each curve, X, Y and Z values are a function of variables u and v whose value ranges are set by user. For example, u is from -50 to 50 and the length of step is 10, v is from 0 to 150 and the length of step is 15, expression of X is cos(u)*sin(v), expression of Y is cos(u)*cos(v) and expression of Z is sin(u). Please note that this plotting tool actually calls a built-in function named plot_3d_surfaces to draw charts. After a chart is plotted, user is able to freely scale, rotate and shift the chart.
