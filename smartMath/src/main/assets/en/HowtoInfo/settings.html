Smart Calculator Help : Settings

Smart Calculator has the following Settings:

1. Bits of precision. This setting determines how many effective bits after decimal points should be shown. For example, if bits of precision is 4, 0.003204876 will be rounded to 0.003205.

2. Scientific notation. This field sets the value range that scientific notation, e.g. 2.1e-37, should be used to show result.

3. Record length. The record length gives out the number of historical calculations recoreded by Smart Calculator.

4. Variable range to plot charts. In smart calculator, if user wants to plot chart for expression(s), variable range is set here. By default, it is from -5 to 5.

5. Vibrate when press calculator button. Check this item if user wants the mobile device to vibrate when press a calculator button.

6. folders' paths. User is able to select which storage is used for the application data folder. There are also two read-only folders' paths. First one is storage path, second is application folder path. These settings will be editable in the future.
